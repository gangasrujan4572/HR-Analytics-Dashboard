# 👥 HR Analytics Dashboard

A Power BI dashboard developed to analyze workforce data and provide actionable insights into employee attrition, demographics, job satisfaction, and organizational performance.

## 📌 Project Overview

This dashboard helps HR professionals monitor key workforce metrics, identify attrition trends, and make data-driven decisions to improve employee retention and organizational effectiveness.

---

## 📊 Dashboard Features

- Employee Overview
- Attrition Analysis
- Department-wise Insights
- Gender Distribution
- Age Group Analysis
- Job Role Distribution
- Education Analysis
- Monthly Income Analysis
- Job Satisfaction Metrics
- Interactive Filters & Slicers

---

## 🛠️ Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX
- CSV Dataset

---

## 📈 Key KPIs

- Total Employees
- Active Employees
- Attrition Count
- Attrition Rate
- Average Age
- Average Salary
- Average Years at Company
